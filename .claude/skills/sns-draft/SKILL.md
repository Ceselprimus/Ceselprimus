# SNS 채널별 문안 생성

## 채널별 규칙 (대표 지시 이력 기반)

| 채널 | 언어 | 스타일 |
|---|---|---|
| 링크드인 (회사 페이지) | **영문만** | 짧고 전문적. B2B·해외 네트워크용. 회사 페이지에 올리고 대표 개인계정이 좋아요/공유 |
| 네이버 블로그 | 한글 | 상세하게. 국내 검색·영업자료용. '쎄슬프라이머스' 한글 표기+해시태그 필수 |
| 페이스북 | 한글 | 링크드인처럼 간결하게 (대표 6/16 지시: "내용 줄이고") |

## 콘텐츠 규칙
- **기술·인사이트 내용만** SNS에 올림. 모델 소개 시리즈는 SNS 금지 (대표 7/5: "장사 속 보임")
- 대표가 문안을 준 경우(docx의 "SNS 업로드용 문구") 그것을 우선 사용하고 채널별로 다듬기만
- 상세 스펙·기술 세부·내부 자료 금지. 개념 수준만
- 이미지: 국문 채널은 `series-N.webp`, 링크드인은 `series-N-en.webp`
- 해시태그 세트:
  - 영문: #AlphaFarm #CESeLPrimus #StrawberryVerticalFarm #VerticalFarming #SmartFarm #AgTech #PremiumStrawberry #ControlledEnvironmentAgriculture (+주제별 2~3개)
  - 한글: #알파팜 #쎄슬프라이머스 #딸기수직농장 #수직농장 #스마트팜 #프리미엄딸기 (+주제별 2~3개)
- 본문 어딘가에 홈페이지 링크(www.ceslprimus.com) 포함 — 검색 신호 + 유입 동선

## 산출물
채널별 3개 문안(제목/본문/해시태그 완성본) + 게시 순서 체크리스트:
1. 네이버 블로그 게시 (한글 이미지)
2. 페이스북 게시 (한글 이미지)
3. 링크드인 회사 페이지 게시 (영문 이미지) → 대표 개인계정 좋아요 요청 카톡
4. 게시 URL을 카톡 보고에 포함

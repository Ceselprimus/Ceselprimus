# GSC 주간 성과 분석

## 입력
GSC → 실적(Performance) → 내보내기로 받은 CSV 묶음 (검색어/페이지/국가/기기/차트 CSV).
이전 주 데이터는 `docs/gsc-history/` 에 보관 (없으면 이번 주 것을 첫 베이스라인으로 저장).

## 분석 항목 (우선순위 순)

1. **타깃 키워드 등장 여부** — 검색어 CSV에서 '딸기수직농장', '딸기 수직농장', '수직농장', 'strawberry vertical farm' 계열 첫 등장/순위 변화. **이것이 홈 중심 SEO 개편의 1차 성공 지표.**
2. **홈페이지(/) 평균순위·노출 추이** — 대표 목표: "딸기수직농장 = 홈페이지"
3. **신규 발행 글 색인 여부** — 최근 2주 발행 글이 페이지 CSV에 등장하는지 (발행 후 ~1주 소요가 정상 패턴)
4. **노출 있는데 클릭 0인 페이지** — 제목·설명 개선 후보
5. **국가별** — 브랜드 재방문(말레이시아형) vs 발견형 유입(기타 국가) 구분. 발견형 증가가 글로벌 인바운드 선행지표
6. **전주 대비 델타** — 클릭/노출/CTR/평균순위

## 산출물
1. 요약 표 + 해석 (전주 대비 화살표)
2. 이번 주 조치 제안 (콘텐츠 발행/제목 수정/색인 요청 등 구체적으로)
3. 대표 보고용 카톡 문구 (숫자 3개 이내로 간결하게)
4. 이번 주 CSV를 `docs/gsc-history/YYYY-MM-DD/`로 복사 보관

## 조치 기준표
| 신호 | 조치 |
|---|---|
| 타깃 키워드 첫 등장 | 해당 페이지 제목·설명 강화 + 관련 글 추가 |
| 노출↑ 클릭 0 | 제목을 클릭 유도형으로 수정 |
| 발행 2주 지나도 색인 안 됨 | GSC URL 검사 → 색인 생성 요청 안내 |
| 순위 4주 정체 | 내부링크 보강 or 콘텐츠 증보 |
